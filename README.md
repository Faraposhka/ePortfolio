# ePortfolio

Simple static ePortfolio. Open `index.html` in your browser to view the site.

Files:

- index.html — main About Me page
- university.html — university, courses, projects
- work.html — experience and research
- other.html — hobbies and links
- styles.css — styling

To run locally: open `index.html` in any browser. For a quick local server (optional):

```powershell
# from c:\PLUS\ePortfolio
python -m http.server 8000
# then open http://localhost:8000 in your browser
```
